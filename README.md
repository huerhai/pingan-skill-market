# 平安健康险技能市场原型

基于OpenClaw平台的平安健康险技能市场原型设计，包含前台展示、后台管理和技能详情页面。

## 项目结构

```
pingan-skill-market/
├── index.html          # 前台首页
├── admin.html          # 后台管理页面
├── skill-detail.html   # 技能详情页面
├── README.md          # 项目说明文档
└── docs/              # PRD文档目录
    └── PRD.md        # 产品需求文档
```

## 功能概述

### 前台功能
1. **首页展示**
   - 技能搜索功能
   - 热门分类展示
   - 推荐技能展示
   - 用户注册/登录

2. **技能市场**
   - 技能分类浏览
   - 技能详情查看
   - 技能下载
   - 用户评价系统

3. **技能上传**
   - 技能信息填写
   - 文件上传
   - 提交审核

### 后台管理功能
1. **技能管理**
   - 技能列表查看
   - 技能审核（通过/拒绝/需要修改）
   - 技能编辑/删除
   - 内测人员管理

2. **用户管理**
   - 用户列表
   - 权限管理
   - 行为日志

3. **数据统计**
   - 下载量统计
   - 用户活跃度
   - 技能评分统计

## 设计特点

1. **响应式设计** - 支持PC、平板、手机访问
2. **交互式原型** - 包含完整的用户交互流程
3. **中国化设计** - 符合国内用户使用习惯
4. **保险行业特色** - 针对保险行业需求定制

## 技术栈

- HTML5 + CSS3 + JavaScript
- Font Awesome图标库
- 原生JavaScript实现交互
- 响应式布局设计

## 使用说明

1. 直接打开 `index.html` 查看前台页面
2. 打开 `admin.html` 查看后台管理页面
3. 打开 `skill-detail.html` 查看技能详情页面

## 部署到GitHub Pages

1. 创建GitHub仓库
2. 将项目文件推送到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源

## 后续开发建议

1. **前端框架** - 建议使用Vue.js或React进行重构
2. **后端开发** - 需要开发RESTful API接口
3. **数据库设计** - MySQL + Redis缓存
4. **安全加固** - 用户认证、数据加密、API防护
5. **移动端APP** - 开发原生或混合移动应用
