## 部署说明

### 本地预览
```bash
cd pingan-skill-market
python3 preview.py
# 或
./preview.py
```

然后在浏览器中访问：http://localhost:8000

### GitHub Pages部署
1. 在GitHub创建新仓库：`pingan-skill-market`
2. 将项目推送到仓库：
   ```bash
   git remote add origin https://github.com/你的用户名/pingan-skill-market.git
   git push -u origin main
   ```
3. 运行部署脚本：
   ```bash
   ./deploy.sh
   ```
4. 按照脚本提示完成GitHub Pages设置

### 访问地址
部署完成后，网站可通过以下地址访问：
```
https://你的用户名.github.io/pingan-skill-market/
```

## 给研发老师的建议

### 技术栈建议
1. **前端**：Vue.js 3 + TypeScript + Element Plus
2. **后端**：Node.js + Express 或 Python + FastAPI
3. **数据库**：MySQL + Redis缓存
4. **部署**：Docker + Kubernetes
5. **监控**：Prometheus + Grafana

### 开发优先级
1. **第一阶段**：用户系统 + 基础技能管理
2. **第二阶段**：完整审核流程 + 文件上传
3. **第三阶段**：数据统计 + 高级功能
4. **第四阶段**：性能优化 + 安全加固

### 注意事项
1. **OpenClaw兼容性**：确保技能包格式兼容
2. **文件安全**：技能包病毒扫描
3. **数据隐私**：用户数据加密存储
4. **合规要求**：符合保险行业监管要求
